# 🚀 BeTalent - API de Gestão de Pagamentos

Esta API foi desenvolvida para o desafio técnico da BeTalent. O sistema gere transações financeiras e clientes, utilizando uma infraestrutura moderna e segura.

## 🛠️ Tecnologias e Diferenciais
- **Framework:** AdonisJS 6 (TypeScript)
- **Base de Dados:** MySQL (Produção)
- **Cloud:** Railway (Base de dados alojada na nuvem)
- **Autenticação:** JWT (Access Tokens)
- **ORM:** Lucid (Uso de `preload` para performance em relacionamentos)

## 📌 Funcionalidades Entregues
- **Nível 1:** Autenticação de utilizadores e geração de tokens.
- **Nível 2:** Listagem de transações com detalhes do Gateway e Utilizador.
- **Nível 3:** Gestão e listagem de base de clientes.
